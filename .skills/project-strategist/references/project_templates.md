# Project Templates

Common project structures and templates for different types of projects.

## Software Development Project

```markdown
# [Project Name] - Software Development

## Overview
- Type: [Web App / Mobile App / API / etc.]
- Language/Framework: [Specify]
- Deployment: [Cloud / On-premise / etc.]

## Phases
1. Requirements & Design (2-4 weeks)
2. Development (4-8 weeks)
3. Testing & QA (2-3 weeks)
4. Deployment & Launch (1 week)

## Key Deliverables
- Functional specification document
- Technical architecture diagram
- Working software
- Test cases & results
- Deployment documentation
```

## Research Project

```markdown
# [Project Name] - Research Study

## Overview
- Domain: [Science / Business / Social / etc.]
- Methodology: [Quantitative / Qualitative / Mixed]
- Duration: [Timeframe]

## Phases
1. Literature Review (2-3 weeks)
2. Research Design (1-2 weeks)
3. Data Collection (2-4 weeks)
4. Analysis (2-3 weeks)
5. Reporting (1-2 weeks)

## Key Deliverables
- Research proposal
- Data collection instruments
- Raw & processed data
- Analysis results
- Final report / publication
```

## Business Process Improvement

```markdown
# [Project Name] - Process Improvement

## Overview
- Current Process: [Brief description]
- Pain Points: [List key issues]
- Expected Improvement: [Quantify if possible]

## Phases
1. Current State Analysis (2-3 weeks)
2. Future State Design (2-3 weeks)
3. Implementation Planning (1-2 weeks)
4. Rollout & Training (2-4 weeks)
5. Monitoring & Optimization (Ongoing)

## Key Deliverables
- Process maps (current & future)
- Gap analysis report
- Implementation plan
- Training materials
- Performance metrics dashboard
```
